# [Matrix Test Task](https://natalia646.github.io/memcrab-test/)

## ‚ú® Features

- Generates a matrix based on the specified number of rows and columns  
  (inputs are limited ‚Äî if a user enters a value higher than the allowed maximum, it is automatically replaced with the maximum allowed value)
- The **X value** is dynamically recalculated according to the selected number of rows and columns
- Ability to add a new row
- Ability to delete any row
- Calculates the total sum for each row
- Calculates the top 60% values for each column
- On cell hover, highlights the nearest **X** values
- On cell click:
  - Increments the value by 1
  - Recalculates row sums
  - Recalculates 60% column values
- On hovering over a row sum cell, related cells are highlighted (Cell Highlighting)
